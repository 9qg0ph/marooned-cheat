# 我独自生活 Frida 快速使用指南

## 🚀 立即开始

### 1. 启动游戏
确保"我独自生活"游戏正在运行

### 2. 运行Frida脚本
```bash
# 在电脑上打开命令行，运行：
frida -U "我独自生活" -l frida_woduzishenghua.js

# 或者使用包名：
frida -U -f "com.Hezi.project1" -l frida_woduzishenghua.js
```

### 3. 查看Hook状态
脚本运行后应该显示：
```

🎯 目标现金数值: 2099999100
[WDZ] 开始初始化Hook...
[WDZ] ✅ integerForKey Hook已安装
[WDZ] ✅ objectForKey Hook已安装
[WDZ] 🎉 所有Hook已安装完成！
```

## 🔍 调试步骤

### 第一步：确认Hook工作
1. 运行脚本后，在游戏中进行操作（买东西、使用体力等）
2. 观察Frida控制台是否有拦截日志
3. 如果有日志显示"🎯 拦截xxx字段"，说明Hook正在工作

### 第二步：监控数值
```javascript
// 在Frida控制台运行
enableMonitor()
```
然后在游戏中操作，查看都读取了哪些数值

### 第三步：精确定位
如果发现游戏读取了你的现金数值2099999100，但没有被拦截，说明需要调整Hook策略

## 🎮 测试流程

1. **启动脚本** → 2. **在游戏中消费** → 3. **查看是否拦截** → 4. **数值是否变化**

## 📱 常见情况

### 情况1：有拦截日志，数值立即变化
✅ 成功！Hook工作正常

### 情况2：有拦截日志，但数值没变化
可能原因：
- 游戏有缓存机制
- 需要触发特定操作刷新界面
- 服务器验证

### 情况3：没有拦截日志
可能原因：
- 游戏不使用NSUserDefaults
- 需要Hook其他方法
- 数值存储在其他地方

## 🔧 如果没效果

尝试以下方法：

1. **监控模式**
```javascript
enableMonitor()
```

2. **更新当前数值**
```javascript
updateCurrentCash(你看到的最新现金数值)
```

3. **重新启动**
- 关闭游戏
- 重新运行Frida脚本
- 启动游戏

这个脚本专门针对你的现金数值2099999100进行了优化！
