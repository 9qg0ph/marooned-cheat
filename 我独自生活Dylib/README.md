### GitHub Actions 自动编译
推送代码到 GitHub 后会自动编译，下载 Artifacts 中的 dylib 文件。

### 手动编译
```bash
clang -arch arm64 \
  -isysroot $(xcrun --sdk iphoneos --show-sdk-path) \
  -miphoneos-version-min=14.0 \
  -dynamiclib \
  -framework UIKit \
  -framework Foundation \
  -fobjc-arc \
  -o WoduziCheat.dylib \
  WoduziCheat.m

ldid -S WoduziCheat.dylib
```
