# 🏠 我独自生活修改器

## 📋 项目说明

一个dylib搞定所有！包含修改器功能，解决闪退问题。

### 🎯 核心文件

- **`WoduziCheat.m`** - 主要修改器（安全版本，不闪退）
- **`WoduziCheat_Full.m`** - 完整版本（功能更多，可能闪退）
- **`Makefile`** - 编译配置

### 🚀 使用方法

```bash
# 编译（推荐使用安全版本）
make package

# 如果需要完整功能，修改Makefile中的文件名为WoduziCheat_Full.m
```
