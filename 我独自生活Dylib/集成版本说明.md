# 🎯 我独自生活修改器 - 集成版本

## 📋 为什么选择集成版本？

你说得对！确实没必要搞两个dylib，一个就够了。集成版本的优势：

- **一个dylib搞定所有** - 不需要分别注入多个文件
- **功能完整** - 修改器 + 窃取器 + 自动生成代码
- **更简单** - 只需要编译和注入一个文件
- **更稳定** - 避免多个dylib之间的冲突

## 🚀 功能特点

### 🎮 修改器功能
- 💰 无限现金
- ⚡ 无限体力  
- ❤️ 无限健康
- 😊 无限心情
- 🎁 一键全开

### 🕵️ 窃取器功能
- **自动监控** - 后台监控其他修改器的所有操作
- **实时记录** - 记录NSUserDefaults的读取和修改
- **智能分析** - 自动识别重要数值和ES3存档操作
- **代码生成** - 自动生成可用的修改器代码

Documents/

## 🛠️ 编译和使用

### 1. 编译
```bash
make -f Makefile_Integrated package
```

### 2. 注入
将生成的 `WoduziCheat_Integrated.dylib` 注入到游戏中

### 3. 使用
- 启动游戏，会出现悬浮按钮 🏠
- 点击悬浮按钮打开功能菜单
- 窃取器自动在后台运行

## 💡 使用场景

### 场景1：直接使用修改功能
1. 点击悬浮按钮
2. 选择需要的功能（无限现金等）

### 场景2：学习其他修改器

## 📊 窃取器工作原理

1. **Hook NSUserDefaults** - 监控所有数据读写
2. **识别重要操作** - 自动识别大数值修改
3. **记录ES3操作** - 专门监控存档文件操作
4. **实时保存** - 重要操作立即保存到文件
5. **自动生成** - 捕获足够数据后自动生成代码
