# 🚀 我独自生活修改器 - 快速开始指南

## 📋 版本选择

### 🎯 v16.0 高级版 (推荐高级用户)
- **特点**: 基于PlayGearLib.dylib技术，多层Hook系统
- **优势**: 自动拦截，智能识别，实时统计
- **适用**: 有iOS逆向经验的用户
- **文件**: `WoduziCheat_Advanced.m`

### 🛡️ v15.3 稳定版 (推荐新手)
- **特点**: 手动修改指导，绝对不闪退
- **优势**: 100%安全，详细教程
- **适用**: 新手用户，追求稳定
- **文件**: `WoduziCheat.m`

## ⚡ 5分钟快速上手

### 方案A: v16.0 高级版

1. **编译dylib**
   ```bash
   cd 我独自生活Dylib
   ./build_advanced.sh
   ```

2. **注入游戏**
   - 使用Sideloadly等工具注入 `WoduziCheat_Advanced.dylib`

3. **启动使用**
   - 游戏中看到红色"高级"按钮
   - 点击 → 启用高级Hook → 启用内存拦截
   - 设置数值 → 在游戏中操作触发拦截

### 方案B: v15.3 稳定版

1. **编译dylib**
   ```bash
   cd 我独自生活Dylib
   clang -arch arm64 -isysroot $(xcrun --sdk iphoneos --show-sdk-path) -miphoneos-version-min=14.0 -dynamiclib -framework UIKit -framework Foundation -fobjc-arc -o WoduziCheat.dylib WoduziCheat.m
   ```

2. **注入游戏**
   - 使用注入工具注入 `WoduziCheat.dylib`

3. **配合使用**
   - 游戏中看到蓝色"独"字按钮
   - 点击 → 查看修改指导
   - 配合iGameGod按指导手动修改

## 🔧 常见问题

### Q: 选择哪个版本？
- **新手**: 选择v15.3稳定版，安全可靠
- **高级用户**: 选择v16.0高级版，功能强大

### Q: v16.0没有效果怎么办？
1. 确保先启用Hook系统
2. 在游戏中进行操作（购买、使用体力等）
3. 查看拦截状态确认Hook是否工作

### Q: v15.3如何使用？
1. 按照日志指导使用iGameGod
2. 先搜索金钱地址
3. 利用偏移关系找其他数值

### Q: 日志在哪里查看？
- **v16.0**: `/var/mobile/Documents/woduzi_advanced.log`
- **v15.3**: `/var/mobile/Documents/woduzi_cheat.log`
